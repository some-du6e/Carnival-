# Carnival+
A chrome extension that does some extra stuff to the [Carnival](https://carnival.hackclub.com/) YSWS.
## What it adds
- projects.js
    - Sums up all the projects
    - Changes the add project button to the new project page
- project.js
    - Checks for placeholders before submitting it for shipping
- qol.js
    - Removes the "intresting" header
- sidebar.js
    - Gets rid of another intresting header in the sidebar
- topbar.js
    - Removes the old ones and lists these:
        - Tokens
        - Hours
        - Dollars (toggable between predicted and real ones)
- utils.js
    - Sends a message to all the modules to trigger them
